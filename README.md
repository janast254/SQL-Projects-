# SQL-Projects-
A few MYSQL projects I have worked on.

Asset_production

I built this SQL database from scratch to track 2,000+ production assets. I calculated 'net loss' to reveal hidden financial leaks and used grouping to see if losses came from many small errors or single 'money pits.' This report gives a clear view of global performance, helping teams focus on the most expensive problems first.


Business Questions Answered 📈
Regional Impact: Which countries are generating the most revenue and where are the highest total losses occurring?

Asset Density: How many active assets are being managed in each specific region?

Loss Patterns: Is a country's high net loss caused by a few "money pits" (single massive failures) or many small, widespread issues?

Risk Extremes: What are the best and worst-case loss scenarios within each country?

Operational Efficiency: What is the average financial loss per active asset across different global markets?
